Source Code

Standalone tools and services

lorestore

https://github.com/uq-eresearch/lorestore/
License: GPL 3.0
lorestore provides services for storing, searching and validating Open Annotation (OA) annotations and ORE Resource Maps.

Annotator

https://github.com/uq-eresearch/annotator/
License: GPL 3.0 or MIT
Annotator is a web annotation system developed by OKFN. We have extended Annotator via plugins to support the Open Annotation model using lorestore, image annotation, custom motivations and replies.

Calliope

https://github.com/AustESE-Infrastructure/calliope
License: GPL 2.0
Calliope provides Multi-Version-Document services.

Services built on Drupal

austese_annotations Drupal Module

https://github.com/uq-eresearch/austese_annotations/
License: GPL 3.0
This module provides a search interface, including keyword and advanced searches over the Annotations stored in lorestore and integrates the Annotator tool and custom plugins into the Workbench.

austese_repository Drupal Module

https://github.com/uq-eresearch/austese_repository/
License: GPL 3.0
This module provides search, display and editing UI and lightweight REST API wrapper around MongoDB for storing metadata and resources within the Content Repository. It also provides import, export, editing and organisation tools for images, transcriptions and other digital resources.

austese_alignment Drupal Module

https://github.com/uq-eresearch/austese_alignment/
License: GPL 3.0
This module provides a custom UI for attaching OA annotations stored in lorestore to record alignment information between images and transcriptions.

austese_collation Drupal Module

https://github.com/uq-eresearch/austese_collation/
License: GPL 3.0
This module provides collation tools for identifying and visualising variants using the Calliope service.

austese_lightbox Drupal Module

https://github.com/uq-eresearch/austese_lightbox/
License: GPL 3.0
This module provides tools for comparing and manipulating images.

AustESE Drupal theme

https://github.com/uq-eresearch/austese_theme/
License: GPL 2.0 or later
Based on Twitter Bootstrap, this is the default theme developed for the AustESE Workbench.

AustESE: Australian Electronic Scholarly Editing

The University of Queensland is proud to be in partnership with the National eResearch Collaboration Tools and Resources (NeCTAR) project to create a unique opportunity to develop eResearch Tools that support the Collaborative Authoring and Management of Electronic Scholarly Editions. This project will benefit the Australian research community by providing an online research and publishing platform that contributes to the preservation and understanding of literary, classical, theological and philosophical texts that have shaped our cultural heritage.
